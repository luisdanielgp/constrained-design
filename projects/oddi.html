Oddi

Developers

LD Guzmán / Vidal Chávez / Nohemi Barrera / Miguel Calderón

Year

2019

Oddi is a web app that allows businesses to automate recurrent payments from clients by using credit/debit card. The project is live and operating since 2019.

The app

The first iteration of the project was a general solution for automating payments. The second iteration was tweaked for NGO's to receive recurrent donations.

As a junior developer in the company my participation in building the back-end of this app from scratch encompassed the following:

  • Entity relations modelling
  • Endpoint's design and implementation
  • Unit testing
  • Invoicing API integration
  • Asincronous tasks

The technologies and tools used for this project were the following:

  • Django
  • DRF
  • Github
  • Docker
  • Celery
